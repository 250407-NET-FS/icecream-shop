Welcome to Cart's Icecream Shop

Cart's Icecream Shop is an app where customers can order a personalized cone or bowl of icecream. Designed for icecream parlors, this app helps to uncover consumer trends to help drive business planning and sales efficiency.

Key Features

  • Custom Ice Cream Pricing: The system allows for setting prices for custom-made ice cream based on ingredients and options.
  • Transaction Management: Cashiers can create and record transactions based on customer orders, facilitating efficient purchase processing.
  • Cashier Order Entry: Cashiers can input customer orders into the system to initiate the transaction process.

How and Where to Access the Shop

  • From the command line (in a terminal), go to the API directory and perform a dotnet run
  • Go to the listed site with port in your browser and append /swagger to the url
  • Try and execute from the several endpoints with inputs grouped by the following models:
    • /icecreams
    • /cashiers
    • /customers
    • /transactions
